### About Me

I am a Ph.D. student at **[AI Game Lab](https://zgc-webcompetition.gitee.io/aigame-lab/)**, jointly trained by [ZGC Academy](https://bjzgca.edu.cn/) and [SUSTech](https://sustech.edu.cn/), supervised by **Prof. [Jian Zhao](https://jianzhao-zgci.github.io/)**.

My research focuses on **AI for Games** ‚Äî building next-generation game engines powered by generative AI, with emphasis on real-time neural rendering and video generation acceleration. Prior to my PhD, I spent 1.5 years as a GPU system algorithm engineer optimizing CUDA kernels and deploying AIGC models on domestic GPU chips.
